## Getting Started

### Development

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The website will open automatically in your browser at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

This will create an optimized build in the `dist/` folder.

## Deployment to Vercel

1. Push your code to GitHub (or your preferred Git provider)

2. Go to [Vercel](https://vercel.com) and import your repository

3. Vercel will automatically detect the Vite configuration and deploy your site!

Alternatively, you can use the Vercel CLI:
```bash
npm i -g vercel
vercel
```
